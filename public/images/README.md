# Изображения сайта

Поместите здесь изображения для сайта:

## Требуемые изображения:

### Hero Section
- `hero.jpg` - Героическое изображение

### About Section
- `about.jpg` - Фото для секции "О нас"

### Gallery
- `gallery-1.jpg` до `gallery-6.jpg` - Фотографии портфолио

## Управление изображениями через TinaCMS

Все остальные изображения (для услуг, мастеров, галереи) управляются через TinaCMS админ-панель. 
Загружайте их там, и они будут сохранены в папке `src/assets/` или `public/uploads/`.

## Размеры рекомендуемых изображений:

- Hero: 1920x1080 или больше
- About: 800x1000
- Gallery: 600x600
- Masters: 400x500
- Services: 400x300
