## ⚠️ Как избежать этого в будущем

### ❌ ПРОБЛЕМЫ:
- ❌ **Кириллица в имени файла** (Безымянный, фото, услуга.png)
- ❌ **Спецсимволы** (файл #1.png, изображение (1).png)
- ❌ **Пробелы в имени** (мое фото.png)

### ✅ РЕШЕНИЕ:
**Перед загрузкой в TinaCMS переименуйте файл на АНГЛИЙСКИЙ язык!**

#### Примеры правильных имен:
```
✓ service-image.png
✓ eyebrows-permanent.jpg
✓ anna-petrova-photo.jpg
✓ nails-design-01.png
```

#### Примеры НЕПРАВИЛЬНЫХ имен:
```
✗ Безымянный.png (кириллица)
✗ фото мастера.jpg (кириллица + пробелы)
✗ image (1).png (пробелы и скобки)
✗ услуга#1.jpg (спецсимволы)
```

## 📝 Пошаговая загрузка изображения

1. **На вашем компьютере:**
   - Найдите фотографию
   - Переименуйте на английский язык
   - Пример: `service-botox.png`

2. **В TinaCMS админ-панели:**
   - Откройте http://localhost:4321/admin
   - Выберите услугу/мастера
   - Нажмите на поле изображения
   - Нажмите **"Upload"**
   - Выберите файл с АНГЛИЙСКИМ названием
   - Нажмите **"Save"**

3. **Проверьте результат:**
   - Откройте http://localhost:4321/services
   - Должны видеть загруженное изображение ✓
