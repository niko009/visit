1\. Стек технологий
-------------------

-   **Framework:** [Astro](https://astro.build/) (SSG --- статический сайт).

-   **Стили:** [Tailwind CSS](https://tailwindcss.com/) (для быстрой и чистой верстки).

-   **Язык:** TypeScript.

-   **Локализация:** `i18next` для Astro.

-   **Контент:** JSON-файлы для локализации и контента (для простоты перевода).

-   **Деплой:** Vercel.

* * * * *

2\. Структура проекта (Архитектура)
-----------------------------------

### Файловая структура (src/)

Plaintext

```
src/
├── components/       # Переиспользуемые UI-блоки (Header, Footer, Button)
├── layouts/          # Основные шаблоны (MainLayout.astro)
├── pages/            # Маршруты сайта (.astro файлы)
├── content/          # Данные (JSON-файлы для текстов и SEO)
├── styles/           # Глобальные CSS-стили
└── i18n/             # Локализация (ru.json, it.json)

```

* * * * *

3\. Требования к реализации страниц
-----------------------------------

ИИ-агент должен создать следующие файлы в `src/pages/`:

1.  `index.astro` --- Главная страница.

2.  `services.astro` --- Список услуг.

3.  `about.astro` --- Наши мастера / О нас.

4.  `testimonials.astro` --- Отзывы.

5.  `gallery.astro` --- До/После.

6.  `blog.astro` --- Список статей, `[slug].astro` для самих статей.

7.  `contacts.astro` --- Контакты.

8.  `privacy.astro` --- Политика конфиденциальности.

9.  `cookies.astro` --- Куки-политика.

* * * * *

4\. Локализация (Одна точка правки текста)
------------------------------------------

Все тексты должны быть вынесены в файлы JSON, чтобы легко переключаться между языками.

**Пример структуры файла `src/i18n/ru.json`:**

JSON

```
{
  "nav": {
    "home": "Главная",
    "services": "Услуги",
    "about": "О нас"
  },
  "hero": {
    "title": "Профессиональный перманентный макияж",
    "cta": "Записаться"
  }
}

```

*ИИ должен использовать `i18next` для вставки этих строк во все компоненты вместо хардкода.*

* * * * *

5\. Организация стилей
----------------------

Используй **Tailwind CSS**. Это стандарт индустрии для современных проектов на Astro.

-   Создай файл `tailwind.config.mjs` для хранения цветовой палитры бренда.

-   Используй компоненты Astro для разбивки логики:

    -   `Button.astro` --- стили кнопки.

    -   `Section.astro` --- обертка для всех блоков контента (с отступами).
